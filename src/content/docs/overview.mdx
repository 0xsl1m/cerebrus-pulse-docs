# Cerebrus Pulse

**Real-time multi-timeframe technical analysis for 50+ Hyperliquid perpetuals, paid via x402 micropayments on Base or Solana.**

Cerebrus Pulse is an API that delivers institutional-grade technical indicators, derivatives data, market regime detection, cross-exchange divergence signals, and multi-signal confluence scoring — all accessible with a single HTTP request and a USDC micropayment.

## Key Features

- **7 indicator families** — RSI, EMAs (20/50/200), ATR, Bollinger Bands, VWAP, Z-score, trend detection
- **Multi-signal confluence** — Automated scoring (0.0–1.0) combining all indicators into a directional bias
- **Derivatives data** — Funding rates, open interest, impact bid/ask, spread (bps)
- **Market regime** — Current regime classification (RISK_ON, RISK_OFF, etc.)
- **Chainlink basis analysis** — HL perpetual oracle vs Chainlink aggregated spot price, measuring exchange premium/discount
- **USDC collateral health** — Real-time depeg monitoring via Chainlink oracle with risk classification
- **Market Stress Index** — Cross-chain arbitrage-derived stress signal (LOW/MODERATE/HIGH/EXTREME)
- **CEX-DEX divergence** — Real-time price comparison across centralized and decentralized exchanges for 13 tokens
- **6 timeframes** — 5-minute, 15-minute, 1-hour, 4-hour, daily, weekly analysis
- **50+ coins** — Major Hyperliquid perpetuals (BTC, ETH, SOL, and more)
- **Dual-chain payments** — Pay with USDC on Base or Solana via x402

## Payment Model

Pulse uses the [x402 protocol](https://www.x402.org/) for payments:

1. You make a standard HTTP request
2. The server responds with `402 Payment Required` and payment details
3. The x402 SDK in your client automatically signs a USDC micropayment on **Base** or **Solana**
4. The server verifies payment and returns the data

**No API keys. No subscriptions. No accounts.** Your wallet is your identity. Pay from whichever chain you prefer.
