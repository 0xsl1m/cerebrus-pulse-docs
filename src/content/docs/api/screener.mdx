# GET /screener

<Badge text="$0.06 USDC" variant="tip" />

Scans all available Hyperliquid perpetuals and returns a compact summary for each coin ‚Äî signal strength, RSI zone, trend direction, volatility regime, funding bias, confluence, and open interest trend. Ideal for market-wide scanning before drilling into individual coins.

## Request

```
GET https://api.cerebruspulse.xyz/screener
```

### Query Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| `top_n` | integer | `50` | 1‚Äì50 | Maximum number of coins to return. |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `timestamp_iso` | string | ISO 8601 timestamp |
| `coins_scanned` | integer | Number of coins analyzed |
| `coins` | array | Per-coin summary objects |
| `coins[].coin` | string | Coin ticker |
| `coins[].signal_strength` | number | Overall signal strength (0.0‚Äì1.0) |
| `coins[].rsi_zone` | string | RSI zone: `OVERSOLD`, `NEUTRAL`, `OVERBOUGHT` |
| `coins[].trend` | string | Trend direction: `BULLISH`, `BEARISH`, `NEUTRAL` |
| `coins[].vol_regime` | string | Volatility regime: `LOW`, `NORMAL`, `HIGH`, `EXTREME` |
| `coins[].funding_bias` | string | Funding bias: `LONG_PAYING`, `SHORT_PAYING`, `NEUTRAL` |
| `coins[].confluence_bias` | string | Confluence bias: `BULLISH`, `BEARISH`, `NEUTRAL` |
| `coins[].oi_trend` | string | OI trend: `RISING`, `FALLING`, `FLAT` |
| `coins[].cex_dex_spread_bps` | number | CEX-DEX price divergence in basis points (where available) |
| `coins[].cex_dex_direction` | string | `cex_premium`, `dex_premium`, or `null` |
| `meta` | object | Provider metadata |

GET /screener?top_n=5
